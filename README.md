## Setup and Deployment
1. **Clone the Repo**: `git clone https://github.com/yourusername/your-repo-name.git`
2. **Edit Locally**: Open `index.html` in a text editor (e.g., VS Code) to customize content, add images (replace src in <img> tags with your URLs), or update the Formspree endpoint in the <form> action.
3. **Host on GitHub Pages**:
   - Push changes to the main branch.
   - Go to Repo Settings > Pages > Set source to main branch > Save.
   - Site live at: https://yourusername.github.io/your-repo-name/
4. **Form Configuration**: Sign up at [formspree.io](https://formspree.io) and replace the form action with your endpoint to enable email submissions.
5. **Adding Images**: Replace placeholder srcs (e.g., https://via.placeholder.com/...) with direct URLs from stock sites like Unsplash.
